  <action>
    Create the subtitle generation service layer:

    1. `editor/src/lib/ai/utils/word-timing.ts` - Word timing utilities:
       - Export interface `WordTiming { word: string; start: number; end: number; confidence: number }`
       - Export interface `SubtitleCue { id: number; start: number; end: number; text: string; words: WordTiming[]; mode: 'karaoke' | 'phrase' }`
       - Export function `groupWordsIntoCues(words: WordTiming[], options: { maxDuration?: number; maxWords?: number; mode: 'karaoke' | 'phrase' }): SubtitleCue[]`
         - For 'phrase' mode: group words into cues of ~5 seconds or until sentence boundary (period, exclamation, question mark). Each cue contains all words for that phrase.
         - For 'karaoke' mode: same grouping logic, but each cue includes per-word start/end timestamps for highlighting
         - Sentence boundary detection: check if word ends with `.`, `!`, `?`
         - Word gap detection: if gap between consecutive words > 1.5s, start new cue
         - Apply 50ms offset adjustment to word timestamps (per research: timing drift compensation)
         - Return array of SubtitleCue sorted by start time

    2. `editor/src/lib/ai/utils/webvtt-generator.ts` - WebVTT and clip data generation:
       - Export function `generateWebVTT(cues: SubtitleCue[]): string`
         - Outputs valid WebVTT format: `WEBVTT\n\n` header
         - Each cue: `{id}\n{start} --> {end}\n{text}\n\n`
         - Time format: `HH:MM:SS.mmm`
         - For karaoke cues, embed word timings in WebVTT metadata comment: `NOTE words={JSON.stringify(cue.words)}`
       - Export function `generateSubtitleClipData(cues: SubtitleCue[], preset: SubtitlePreset, videoWidth: number, videoHeight: number): any[]`
         - Converts SubtitleCues into clip JSON objects compatible with the existing Caption clip format (matching `caption-generator.ts` output shape)
         - Each clip includes: type 'Caption', display timing (from/to in microseconds), position (from preset defaults), text, style (from preset), caption.words (for karaoke highlighting), caption.colors (from preset)
         - For karaoke mode: set `wordsPerLine: 'multiple'` and include word timing data in caption.words array
         - For phrase mode: set `wordsPerLine: 'multiple'` with no per-word highlighting
         - Position calculation uses preset.position ('top', 'center', 'bottom') mapped to videoHeight coordinates with padding

    3. `editor/src/lib/ai/services/subtitle-service.ts` - Service orchestrator:
       - Export class `SubtitleService`
       - Method `generateFromAudio(audioUrl: string, options: { mode: 'karaoke' | 'phrase'; preset: SubtitlePreset }): Promise<SubtitleCue[]>`
         - Calls existing `transcribe({ url: audioUrl, words: true, model: 'nova-3' })` from `@/lib/transcribe`
         - Extracts word-level timestamps from result: `result.results.main.words`
         - Calls `groupWordsIntoCues(words, { mode: options.mode })`
         - Returns SubtitleCue[]
       - Method `generateFromScript(script: string, ttsAudioUrl: string, options: { mode: 'karaoke' | 'phrase'; preset: SubtitlePreset }): Promise<SubtitleCue[]>`
         - When TTS voiceover exists and we have the script text: transcribe the TTS audio to get word timestamps
         - Calls `generateFromAudio(ttsAudioUrl, options)` (forced alignment through transcription)
         - Returns SubtitleCue[]
       - Method `detectSource(clips: Array<{ type: string; src?: string }>): 'tts' | 'audio'`
         - Check if any clip is an Audio clip with a voiceover URL (contains 'voiceovers/' in path)
         - If yes: return 'tts' (use script-based generation)
         - If no: return 'audio' (transcribe existing audio/video)
       - Static method `getDefaultPreset(): SubtitlePreset` returns 'modern-karaoke' preset
  </action>

    Define subtitle presets and create the subtitle generation API:

    1. `editor/src/lib/ai/presets/subtitle-presets.ts` - Preset definitions:
       - Export interface `SubtitlePreset`:
         ```
         {
           id: string;
           name: string;
           description: string;
           mode: 'karaoke' | 'phrase';
           position: 'top' | 'center' | 'bottom';
           style: {
             fontSize: number;
             fontFamily: string;
             fontWeight: string;
             color: string;
             align: 'center' | 'left' | 'right';
             fontUrl?: string;
             stroke?: { color: string; width: number };
             shadow?: { color: string; alpha: number; blur: number; offsetX: number; offsetY: number };
             background?: string;
           };
           colors: {
             appeared: string;   // color for already-spoken words
             active: string;     // text color for current word
             activeFill: string; // highlight/fill color for current word
             background: string; // background behind text
             keyword: string;    // keyword emphasis color
           };
           preview?: string;  // thumbnail/preview image path
         }
         ```
       - Export `SUBTITLE_PRESETS: SubtitlePreset[]` with 8 presets:
         1. **modern-karaoke**: White text, orange highlight (#FF5700), black stroke, Poppins bold, bottom position, karaoke mode - the default TikTok/CapCut style
         2. **classic-phrase**: White text on semi-transparent black bar, no highlight animation, Inter regular, bottom position, phrase mode - traditional subtitle look
         3. **neon-glow**: Cyan text (#00FFFF) with glow shadow, pink highlight (#FF00FF), no stroke, Orbitron bold, bottom position, karaoke mode
         4. **minimal-clean**: Light gray text, subtle underline highlight, no background, Inter Light, bottom position, phrase mode
         5. **bold-impact**: Yellow text (#FFD700), white highlight, heavy black stroke (width 6), Impact/Bangers bold, center position, karaoke mode
         6. **gradient-pop**: White text, gradient-simulated highlight (bright orange to yellow), Montserrat bold, bottom position, karaoke mode
         7. **typewriter**: Monospace font (JetBrains Mono), green text (#00FF41), no highlight, dark background bar, top position, phrase mode - terminal/hacker aesthetic
         8. **social-viral**: White text with heavy black stroke, red highlight (#FF0000), Poppins ExtraBold, center position, karaoke mode - optimized for vertical video
       - Export helper: `getPresetById(id: string): SubtitlePreset | undefined`
       - Export helper: `getDefaultPreset(): SubtitlePreset` returns 'modern-karaoke'

    2. `editor/src/app/api/ai/subtitles/route.ts` - Subtitle generation endpoint:
       - `POST /api/ai/subtitles` - Generates subtitles from audio
       - Request body:
         ```
         {
           audioUrl: string;          // URL of audio/video to transcribe
           presetId?: string;         // Subtitle preset ID (default: 'modern-karaoke')
           mode?: 'karaoke' | 'phrase'; // Override preset mode
           videoWidth?: number;       // Canvas width for positioning (default: 1920)
           videoHeight?: number;      // Canvas height for positioning (default: 1080)
           format?: 'clips' | 'webvtt'; // Output format (default: 'clips')
         }
         ```
       - Validate: audioUrl required
       - Resolve preset: use presetId to look up preset, override mode if provided
       - Create `SubtitleService` instance and call `generateFromAudio(audioUrl, { mode, preset })`
       - If format='webvtt': return `{ webvtt: string }` using `generateWebVTT(cues)`
       - If format='clips' (default): return `{ clips: any[], cues: SubtitleCue[] }` using `generateSubtitleClipData(cues, preset, videoWidth, videoHeight)`
       - Return 200 with subtitle data, 400 for validation errors, 500 for transcription failures
