  <action>
    Create the LLM-powered template generation service:

    1. `editor/src/lib/ai/utils/merge-field-detector.ts` - Merge field detection:
       - Export interface `MergeFieldSuggestion { elementId: string; property: string; label: string; type: 'text' | 'image' | 'color' | 'video' | 'number' }`
       - Export function `detectMergeFields(elements: TemplateElement[]): MergeFieldSuggestion[]`:
         - Scan each element for properties that should be dynamic:
         - Text elements: mark `text` property as merge field with type 'text', auto-generate label from content (e.g., "Welcome to Our Company" -> "Company Name" or "Heading Text")
         - Image elements: mark `src` property as merge field with type 'image', label based on element name or "Product Image", "Logo"
         - Video elements: mark `src` as merge field with type 'video', label "Background Video"
         - Color properties: if element has explicit brand/accent color (not black/white/gray), mark as merge field with type 'color', label "Brand Color", "Accent Color"
         - Use heuristics: elements with generic placeholder text (containing "Lorem", "Your", "Company", "[", etc.) are more likely merge fields
         - Elements named with obvious placeholders (e.g., "logo", "title", "headline") are merge fields
         - Return array of MergeFieldSuggestion
       - Export function `generateMergeFieldLabel(elementType: string, property: string, value: any): string`:
         - Smart label generation: "Welcome to..." -> "Welcome Text", image with "logo" in name -> "Logo Image"
         - Fallback: capitalize property name

    2. `editor/src/lib/ai/services/template-generation-service.ts` - Generation service:
       - Import Anthropic from '@anthropic-ai/sdk' (installed in Plan 05)
       - Export class `TemplateGenerationService`
       - Constructor takes `{ apiKey: string }`
       - Define the template element schema for Claude tool calling:
         ```
         const TOOL_SCHEMA = {
           name: 'generate_template',
           description: 'Generate a video template as structured JSON',
           input_schema: {
             type: 'object',
             properties: {
               name: { type: 'string' },
               description: { type: 'string' },
               duration: { type: 'number', description: 'Total duration in seconds' },
               settings: {
                 type: 'object',
                 properties: {
                   width: { type: 'number' },
                   height: { type: 'number' },
                   fps: { type: 'number' },
                   backgroundColor: { type: 'string' }
                 }
               },
               elements: {
                 type: 'array',
                 items: {
                   type: 'object',
                   properties: {
                     id: { type: 'string' },
                     type: { type: 'string', enum: ['text', 'image', 'video', 'shape', 'audio'] },
                     name: { type: 'string' },
                     properties: { type: 'object' },
                     position: { type: 'object', properties: { x: { type: 'number' }, y: { type: 'number' }, width: { type: 'number' }, height: { type: 'number' } } },
                     timing: { type: 'object', properties: { start: { type: 'number' }, duration: { type: 'number' } } },
                     animation: { type: 'object', properties: { type: { type: 'string' }, duration: { type: 'number' }, easing: { type: 'string' } } }
                   },
                   required: ['id', 'type', 'name', 'properties', 'position', 'timing']
                 }
               },
               transitions: {
                 type: 'array',
                 items: {
                   type: 'object',
                   properties: {
                     type: { type: 'string' },
                     duration: { type: 'number' },
                     atTime: { type: 'number' }
                   }
                 }
               }
             },
             required: ['name', 'description', 'duration', 'elements']
           }
         }
         ```
       - Build a comprehensive system prompt (stored as const SYSTEM_PROMPT) that:
         - Describes all 12 style presets and their visual characteristics
         - Instructs the model to use placeholder content that clearly identifies merge fields (e.g., "Your Company Name", "[Product Image]", "Brand tagline here")
         - Defines the element property schemas for each type (text: fontSize, fontFamily, color, fontWeight; image: src, objectFit; etc.)
         - Specifies available animations: fadeIn, slideInLeft, slideInRight, slideInUp, scaleIn, etc.
         - Specifies available transitions: fade, dissolve, wipe, slide
         - Instructs: "Generate production-ready templates. Use realistic dimensions (1920x1080 for landscape, 1080x1920 for portrait). Include at least 3 text elements, 1-2 image placeholders, and transitions."

       - Method `generate(prompt: string, styleId: string): Promise<GeneratedTemplate>`:
         - Load style preset from TEMPLATE_STYLE_PRESETS
         - Call `anthropic.messages.create({ model: 'claude-sonnet-4-5', max_tokens: 4096, system: SYSTEM_PROMPT, messages: [{ role: 'user', content: 'Create a {style.name} style video template: {prompt}' }], tools: [TOOL_SCHEMA], tool_choice: { type: 'tool', name: 'generate_template' } })`
         - Extract tool_use result from response, validate with basic type checks
         - Call `detectMergeFields(template.elements)` to auto-detect merge fields
         - Return `{ template, mergeFields, conversationHistory: [messages] }`

       - Method `refine(existingTemplate: any, refinementPrompt: string, conversationHistory: Array<{ role: string; content: any }>): Promise<GeneratedTemplate>`:
         - Build messages array with conversation history + new refinement request
         - Include existing template as assistant context
         - Add instruction: "Modify the existing template based on the user's request. Only change what's requested, preserve everything else."
         - Same tool calling approach as generate()
         - Re-run merge field detection on refined template
         - Return updated template with extended conversation history

       - Export interface `GeneratedTemplate { template: any; mergeFields: MergeFieldSuggestion[]; conversationHistory: Array<{ role: string; content: any }> }`

    2. `editor/src/app/api/ai/template/route.ts` - Template generation endpoint:
       - `POST /api/ai/template` - Generates template from description
       - Request body: `{ prompt: string; styleId: string }`
       - Validate: prompt required (1-500 chars), styleId must match known preset
       - Create `TemplateGenerationService` with `process.env.ANTHROPIC_API_KEY`

       - Generate a conversationId (UUID) and store conversation history in a module-level Map with 30-minute TTL (simple in-memory cache, sufficient for single-server deployment)

       - If not found (expired or invalid): start fresh conversation with currentTemplate as context
       - Call `service.refine(currentTemplate, prompt, conversationHistory)`
       - Update conversation history in Map
       - Return same shape as generation endpoint: `{ template: any; mergeFields: MergeFieldSuggestion[]; conversationId: string }`
