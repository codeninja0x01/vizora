Create editor/src/workers/webhook-worker.ts as a standalone BullMQ worker process (following the pattern from render-worker.ts):

**Imports:**
- `Worker`, `Job`, `UnrecoverableError` from `'bullmq'`
- `request` from `'undici'`
- `prisma` from `'@/lib/db'`
- `redisConnection` from `'@/lib/redis'`
- `generateWebhookSignature` from `'@/lib/webhooks/signature'`
- `WebhookJobData` from `'@/lib/webhooks/types'`

**Job processor function `processWebhookDelivery(job: Job<WebhookJobData>)`:**

1. Extract `webhookConfigId`, `renderId`, `webhookId`, `payload` from `job.data`

2. Fetch webhook config from DB:
   ```typescript
   const config = await prisma.webhookConfig.findUnique({
     where: { id: webhookConfigId }
   });
   ```
   If not found or `!config.enabled`, return early (graceful skip, do NOT throw — the webhook was deleted/disabled while job was in queue)

3. Prepare delivery:
   - `const timestamp = Math.floor(Date.now() / 1000)`
   - `const payloadString = JSON.stringify(payload)`
   - Generate signature: `generateWebhookSignature(webhookId, timestamp, payloadString, config.secret)`

4. Send HTTP POST via undici:
   ```typescript
   const response = await request(config.url, {
     method: 'POST',
     headers: {
       'Content-Type': 'application/json',
       'webhook-id': webhookId,
       'webhook-timestamp': timestamp.toString(),
       'webhook-signature': signature,
       'user-agent': 'OpenVideo-Webhooks/1.0',
     },
     body: payloadString,
     bodyTimeout: 30000,
     headersTimeout: 30000,
     maxRedirections: 0, // Disable redirects for SSRF protection
   });
   ```

5. Handle response:
   - 2xx (200-299): Success — update DB metadata:
     ```typescript
     await prisma.webhookConfig.update({
       where: { id: webhookConfigId },
       data: { lastDeliveryAt: new Date(), lastSuccessAt: new Date(), consecutiveFailures: 0 }
     });
     ```
     Log success. Return.

   - 429 or 5xx (500-599): Retryable — throw regular Error so BullMQ retries with backoff:
     ```typescript
     throw new Error(`Webhook delivery failed: HTTP ${response.statusCode}`);
     ```

   - 4xx (except 429): Permanent failure — throw UnrecoverableError so BullMQ does NOT retry:
     ```typescript
     throw new UnrecoverableError(`Webhook permanently rejected: HTTP ${response.statusCode}`);
     ```

6. Consume response body to prevent memory leaks: `await response.body.dump()` after reading status code.

**Worker configuration:**
```typescript
const worker = new Worker<WebhookJobData>('webhook-delivery', processWebhookDelivery, {
  connection: redisConnection,
  concurrency: 5, // Process 5 webhook deliveries in parallel
  settings: {
    backoffStrategy: (attemptsMade: number): number => {
      // Custom 5^n pattern: 5s, 25s, 125s, 625s, 3125s
      return Math.pow(5, attemptsMade) * 1000;
    },
  },
});
```

**Event handlers (same pattern as render worker):**
- `completed`: Log success
- `failed`: Log failure. If `job.attemptsMade >= job.opts.attempts` (all retries exhausted), update DB:
  ```typescript
  await prisma.webhookConfig.update({
    where: { id: job.data.webhookConfigId },
    data: { lastDeliveryAt: new Date(), lastFailureAt: new Date(), consecutiveFailures: { increment: 1 } }
  });
  ```

**Graceful shutdown (same pattern as render worker):**
- SIGTERM/SIGINT handlers that close worker and disconnect Prisma

**Startup logging:**
```
[Webhook Worker] Webhook delivery worker started, waiting for jobs...
[Webhook Worker] Concurrency: 5
```

**Create a helper function `enqueueWebhooks`:**
```typescript
async function enqueueWebhooks(
  renderId: string,
  organizationId: string,
  payload: WebhookPayload
): Promise<void> {
  try {
    // Find all enabled webhooks for this organization
    const webhooks = await prisma.webhookConfig.findMany({
      where: { organizationId, enabled: true },
      select: { id: true },
    });

    if (webhooks.length === 0) return;

    // Enqueue a delivery job for each webhook
    const jobs = webhooks.map((webhook) => ({
      name: 'deliver-webhook',
      data: {
        webhookConfigId: webhook.id,
        renderId,
        webhookId: `whk_${randomUUID().replace(/-/g, '')}`,
        payload,
      },
    }));

    await webhookQueue.addBulk(jobs);
    console.log(`[Worker] Enqueued ${jobs.length} webhook(s) for render ${renderId}`);
  } catch (error) {
    // Log but do NOT throw — webhook failure must not affect render status
    console.error(`[Worker] Failed to enqueue webhooks for render ${renderId}:`, error);
  }
}
```
