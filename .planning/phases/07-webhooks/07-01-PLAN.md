1. Create editor/src/lib/webhooks/signature.ts:

- Import `createHmac`, `timingSafeEqual`, `randomBytes` from `'node:crypto'`

- `generateWebhookSecret()`: Returns `randomBytes(32).toString('base64')` (256-bit secret)

- `generateWebhookSignature(webhookId: string, timestamp: number, payload: string, secret: string)`:
  - Construct signed string: `${webhookId}.${timestamp}.${payload}`
  - Create HMAC-SHA256 with secret, update with signed string, digest as base64
  - Return `v1,${signature}` (v1 prefix indicates HMAC-SHA256)

- `verifyWebhookSignature(webhookId: string, timestamp: number, payload: string, secret: string, providedSignature: string)`:
  - Check timestamp tolerance: reject if abs(now - timestamp) > 300 seconds (5 min replay protection)
  - Generate expected signature using generateWebhookSignature
  - Convert both expected and provided to Buffers
  - Return false if lengths differ (required for timingSafeEqual)
  - Return `timingSafeEqual(expectedBuffer, providedBuffer)`
  - Wrap in try/catch, return false on any error

2. Create editor/src/lib/webhooks/validator.ts:

- `validateWebhookUrl(urlString: string)`: Returns `{ valid: boolean; error?: string }`
  - Parse URL with `new URL()`, catch invalid formats
  - In production (NODE_ENV === 'production'): require `https:` protocol
  - In development: allow `http://localhost`
  - Block these hostnames: `localhost`, `127.0.0.1`, `0.0.0.0`, `::1`, `169.254.169.254` (AWS metadata), `metadata.google.internal` (GCP metadata)
  - If hostname is an IP (use `net.isIP()`): check against private ranges
  - `isPrivateOrReservedIP(ip: string)`: Check RFC 1918 ranges:
    - 10.0.0.0/8 (parts[0] === 10)
    - 172.16.0.0/12 (parts[0] === 172 && parts[1] >= 16 && parts[1] <= 31)
    - 192.168.0.0/16 (parts[0] === 192 && parts[1] === 168)
    - 127.0.0.0/8 (parts[0] === 127)
    - 169.254.0.0/16 (parts[0] === 169 && parts[1] === 254)
  - NOTE: In development, bypass private IP check and hostname blocklist for `localhost` only (allow testing against local webhook receivers)

3. Create editor/src/lib/webhooks/queue.ts:
