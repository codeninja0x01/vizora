1. Create editor/src/app/(protected)/dashboard/webhooks/actions.ts with `'use server'` directive:

**`createWebhook(url: string)`:**
- Get session via `auth.api.getSession({ headers: await headers() })`
- Validate URL is non-empty and valid format
- Call `validateWebhookUrl(url)` — return `{ error: string }` if invalid
- Generate secret via `generateWebhookSecret()`
- Create via `prisma.webhookConfig.create({ data: { organizationId: session.session.activeOrganizationId, url, secret } })`
- Call `revalidatePath('/dashboard/webhooks')`
- Return `{ id: config.id, secret }` — secret returned once for display

**`deleteWebhook(id: string)`:**
- Get session, verify ownership: findUnique by id where organizationId matches
- Delete via prisma
- Call `revalidatePath('/dashboard/webhooks')`
- Return `{ success: true }`

**`rotateWebhookSecret(id: string)`:**
- Get session, verify ownership
- Generate new secret via `generateWebhookSecret()`
- Update via prisma.webhookConfig.update
- Reset consecutiveFailures to 0
- Call `revalidatePath('/dashboard/webhooks')`
- Return `{ secret }` — new secret returned once for display

**`toggleWebhook(id: string, enabled: boolean)`:**
- Get session, verify ownership
- Update enabled field
- Call `revalidatePath('/dashboard/webhooks')`
- Return `{ success: true }`

- URL input with placeholder "https://example.com/webhooks/openvideo"
- Use Webhook icon from lucide-react

Style following existing dashboard dark theme: bg-white/5, border-white/10, text-foreground, primary accent for actions.
